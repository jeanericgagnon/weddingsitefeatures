# Features

Each feature should live in its own folder:

- `features/<feature-name>/README.md`
- `features/<feature-name>/src/...`
- `features/<feature-name>/INTEGRATION.md`
- optional `features/<feature-name>/migrations/...`

Suggested naming:
- `rsvp-reminders`
- `guest-tags`
- `wedding-timeline-widget`
- `planner-activity-feed`
