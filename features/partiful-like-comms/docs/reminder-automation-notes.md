# Reminder automation notes

## What this batch adds

- reminder rule types
- candidate filtering utilities
- example rule + candidate set

## Why it matters

Reminder automation is one of the highest-leverage comms features because it turns RSVP chasing into a system instead of manual admin work.

## Practical rules to support first

- remind `not_responded` guests 5 days before RSVP deadline
- remind again 2 days before deadline if still pending
- enforce cooldown to avoid spammy double-sends
