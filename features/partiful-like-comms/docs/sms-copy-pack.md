# SMS copy pack

## Billing / plan copy

### Preferred
- `1,000 segments included`
- `Segments remaining`
- `SMS usage`

### Avoid
- `1,000 texts included`
- `Unlimited texting`
- `Per text` when billing is actually segment-based

## Composer meter copy

### Normal
- `142 / 160 characters`
- `1 segment`
- `987 segments remaining`

### Warning
- `181 characters`
- `2 segments ⚠️`

## Soft threshold copy
- `You're running low on text credits`
- `You're using text segments faster than usual`
- `Longer messages and emojis can use multiple segments`

## Failure / review copy
- `Message delivery failed`
- `Carrier rejected destination number`
- `Reply needs review`
- `We couldn't confidently match this reply to a guest`

## Contact capture copy
- `Add your contact details so the couple can send updates`
- `I agree to receive SMS updates about this event`

## Tone notes
- plain English
- direct
- no fake telecom jargon
- honest about cost behavior
